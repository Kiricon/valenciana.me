My tools for success
Javascript
I've been working with javascript for many years now now and love doing front-end web development with it.
I develop a majority of my projects in JQuery because of the ease of use that it brings to DOM manipulation.
When it comes to having a great MVC model front end and keeping maintainable object-oriented code I prefer the Backbone Framework
PHP
For my server side code I prefer php. It's well documented apis, object oriented design and open source BSD-style license makes it's my server side language of choice.
Laravel is one of my MVC frameworks of choice, due to it's massive community and well documented apis.

More commonly however I use the Phalcon MVC framework due to it's #1 server performance speed. Being compiled as a C extension you can't find a faster full functioned MVC framework.
CSS
It goes without saying that you'd need CSS in your webpage. Something that deserves more attention however, is well done 60fps animations that work perfectly with the critical rendering path. And don't even get me started on web pages that don't properly follow modern UI/UX standards or don't have proper mobile compatibility.
